# GadgetZone Ecommerce Website

Welcome to GadgetZone, an ecommerce platform for electronics products!

## Overview

GadgetZone is a full-stack ecommerce website designed to offer a seamless shopping experience for electronics enthusiasts. With a focus on providing a wide range of electronics items and user-friendly features, GadgetZone aims to make online shopping convenient and enjoyable.

## Tech Stack

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: PHP
- **Database**: MySQL

## Features

- **SEO Optimized**: GadgetZone is optimized for search engines, ensuring high visibility and discoverability online.
- **Fast Load Time**: With a focus on performance, GadgetZone ensures that pages load quickly, providing a smooth browsing experience for users.
- **Add to Cart**: Customers can easily add products to their cart for quick and convenient checkout.
- **Admin Panel**: An intuitive admin panel allows administrators to manage products, orders, and users effortlessly.
- **Admin Login**: Secure admin login functionality ensures that only authorized personnel can access the admin panel.
- **User Authentication**: Users can create accounts, login, and signup to access personalized features and track their orders.
- **Inventory Management**: GadgetZone includes robust inventory management features, enabling administrators to track stock levels, update product information, and manage product availability effectively.

## Getting Started

To run the GadgetZone ecommerce website locally on your machine, follow these steps:

1. Clone the repository: `git clone https://github.com/your-username/gadgetzone.git`
2. Set up the database: Configure MySQL connection settings in `config.php`
3. Start a local server (e.g., XAMPP, WAMP) and import the provided MySQL database dump.
4. Access the website: Open your browser and navigate to `http://localhost/gadgetzone`
